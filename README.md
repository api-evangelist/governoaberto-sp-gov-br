# Governo Aberto do Estado de São Paulo (governoaberto-sp-gov-br)

Governo Aberto do Estado de São Paulo is a open data portal open-data portal for br running **CKAN**, exposing a standardized catalog API over ~419 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/governoaberto-sp-gov-br/refs/heads/main/apis.yml)
